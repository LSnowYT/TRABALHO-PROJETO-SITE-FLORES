 Sobre o Projeto

Este projeto consiste em um site simples desenvolvido com HTML e CSS, com foco na apresentação de um site para cobrança. O objetivo principal é praticar conceitos básicos de desenvolvimento front-end, como estruturação de páginas, estilização e organização de conteúdo.

## 🚀 Funcionalidades

* Layout de site estático
* Estilização com CSS
* Estrutura organizada em seções
* Apresentação visual de uma loja de flores

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* java script


## ▶️ Como Executar

1. Clone o repositório:

```
git clone https://github.com/LSnowYT/TRABALHO-PROJETO-SITE-FLORES.git
```

2. Abra o arquivo `index.html` em qualquer navegador.

## 🎯 Objetivo

Este projeto foi desenvolvido com fins educacionais, com o intuito de praticar a criação e estruturação de páginas web.
